# Hero image prompt for fal.ai

## Goal
Create a premium customer-facing fintech hero illustration for a loyalty platform.

## Visual direction
- modern fintech
- friendly, trustworthy, clean
- customer-facing, not corporate dashboard heavy
- elegant, light, optimistic
- simple composition with breathing room

## Main colors
- white
- emerald green
- soft mint green
- subtle neutral gray
- optional accent: soft gold or teal, very restrained

## What should appear
- abstract fintech / loyalty scene
- wallet, benefits, points, progress, rewards, digital account feeling
- no literal brand logos
- no busy stock-photo style people
- if a person appears, use one minimal friendly customer illustration only
- UI-inspired shapes/cards are welcome
- should feel like a modern banking / rewards product

## What to avoid
- dark cyberpunk style
- neon purple/blue dominance
- crypto aesthetic
- overly futuristic sci-fi
- generic AI chaos with too many floating elements
- text inside the image
- watermarks
- cluttered marketing poster composition

## Composition
- horizontal hero image
- suitable for website landing section
- leave clean negative space on one side for headline + CTA overlay if needed
- rounded soft shapes
- soft light and subtle depth

## Final prompt
A premium modern fintech hero illustration for a customer loyalty platform, designed for a clean website landing page. Use a bright white and emerald green palette with soft mint accents and subtle neutral grays. The image should feel friendly, trustworthy, elegant, and customer-centric. Show an abstract visual world inspired by wallet, rewards, loyalty points, progress, benefits, and digital account experience, using refined UI-like cards, soft rounded shapes, subtle depth, and light gradients. Keep the composition clean and minimal with generous breathing room, suitable for a website hero section. Avoid cyberpunk, crypto aesthetics, dark themes, clutter, excessive floating objects, text, logos, and watermarks.

## Suggested negative prompt
Dark background, cyberpunk, purple neon, blue neon, crypto, sci-fi, cluttered composition, too many objects, text, watermark, logo, aggressive contrast, generic stock illustration, low quality, distorted UI, messy scene
